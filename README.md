A Book Scraper Project

A simple Python web scraper that extracts book information (title, price, etc.) from a website and saves the data into a CSV file.

Features
- Fast and lightweight scraping
- Automatically saves results to `books.csv`
- Easy to run with one command

Requirements
- Python 3.8+
- `requests`
- `beautifulsoup4`
